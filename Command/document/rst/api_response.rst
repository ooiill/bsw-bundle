
◈ 接口响应相关
========================================

加密响应
----------------------------------------

1. 通过 ``json_encode`` 对响应数据对象进行字符串化得到 ``$responseText``。

#. 使用 ``AES-128-CBC`` 加密算法对 ``$responseText`` 进行加密后响应到客户端。

.. note::

    客户端得到后对字符串解密，然后再 ``json_decode`` 即可。
